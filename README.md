# File Organizer

Программа для автоматической сортировки файлов по категориям.

## Возможности
- Умная сортировка по расширениям
- Редактор правил и масок
- Безопасный режим
- Поддержка Regex и хеширования
- Визуализация процесса

## Установка
1. Клонировать репозиторий
2. Запустить: python file_organizer.py

## Структура
- file_organizer.py - основной код приложения
- ПИ1.pptx - презентация проекта
- ПИ-ТЗ-1.docx - техническое задание
